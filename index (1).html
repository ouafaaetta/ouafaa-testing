Site structure and minimalist content (With a default global layout)

_config.yml
# Remote theme, use the latest version
remote_theme: wet-boew/gcweb-jekyll

#
# Page front matter defaults
defaults:
  - scope:
      path: "" # Ensure it's applied to all pages
    values:
      layout: default
index.html
